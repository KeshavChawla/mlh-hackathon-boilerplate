FAQ


function what_is_a_hackathon? ( ) {

A hackathon is an invention marathon. Students come together to build cool software & hardware hacks over 24-48 hours. It's very beginner friendly.

}
function how_much_does_it_cost? ( ) {

Nothing. Attending is free. Food and prizes are on us!

}
function do_i_need_to_be_a_student_to_attend? ( ) {

Yes. Only students who are currently enrolled at a highschool or have graduated within the past 12 months are eligible to attend.

}

function ok_if_no_experience? ( ) {

That's completely fine! You'll have plenty of opportunity to pick the brains of talented students around you. Furthermore, we will help you get started or learn something new!

}
function what_to_build? ( ) {

You can build whatever you want: websites, programs, mobile apps, anything! However, all code must be written during the hackathon - feel free to use open source libraries and APIs.

}
function num_people_per_team ( ) {

We recommend working in teams of 3-4 people, though participants in the past have worked solo or in pairs as well. However, if your team would like to submit your project for prize consideration, we have a hard limit of 4 members per team.

}
